Tools & Platforms

Essential security tools, platforms, and software used by professionals in the field.

Penetration Testing

  • Kali Linux

    Security-focused Linux distribution with hundreds of pre-installed security tools

  • Burp Suite

    Integrated platform for web application security testing

  • Metasploit Framework

    Advanced open-source platform for developing, testing, and executing exploits

Defense & Monitoring

  • Snort

    Open source intrusion prevention and detection system

  • Elastic Security

    Free and open SIEM solution for threat detection and incident response

  • Wireshark

    Network protocol analyzer for network troubleshooting and analysis
